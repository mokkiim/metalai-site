# MCP (Model Context Protocol)

> AI 어시스턴트가 외부 데이터·도구와 연결되는 오픈 표준 프로토콜. AI 생태계의 "USB-C"

## 상세 설명
2024년 11월 Anthropic이 발표한 오픈 프로토콜. AI 앱과 외부 도구/데이터 사이의 표준화된 양방향 JSON-RPC 연결을 정의한다.

## 핵심 포인트
1. **표준화**: AI 에이전트가 도구를 호출하는 "공용어"
2. **양방향**: 컨텍스트 요청 + 도구 실행 모두 가능
3. **생태계 폭발**: 5,800+ 서버, 전 업체 채택
4. **2026 로드맵**: 엔터프라이즈 인증, Agent-to-Agent 통신
